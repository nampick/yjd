# Rich Text Editor

A lightweight rich text editor built with vanilla JavaScript, inspired by TinyMCE.

## Features (Planned)

- Basic text formatting (bold, italic, underline)
- Text alignment
- Lists (ordered and unordered)
- Link insertion
- Image upload
- Undo/Redo functionality
- Clean and modern UI

## Project Structure

```
richeditor/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Styles for the editor
├── js/
│   ├── editor.js      # Main editor logic
│   ├── commands.js    # Editor commands implementation
│   └── toolbar.js     # Toolbar functionality
└── README.md          # This file
```

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Start editing!

## Development

This project uses vanilla JavaScript with no external dependencies. To contribute:

1. Fork the repository
2. Create your feature branch
3. Make your changes
4. Submit a pull request
