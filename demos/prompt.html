AI chat input demo — a prompt-style rich text composer

A chat-style composer in the shape of a rounded pill — the same layout as modern AI prompt inputs. The “+” adds an image (a popover of image/file/video on desktop), attachments ride along as removable thumbnails, and Enter sends while the on-screen keyboard’s return key inserts a newline on mobile. Built for AI prompts, messaging and comment composers.
