Bubble menu rich text editor demo — floating toolbar on select

A distraction-free writing surface with no top toolbar. Select any text and a floating “bubble” bar appears with the formatting you need; type / for a block menu (headings, lists, and more). Great for Medium- or Notion-style editors where chrome should stay out of the way.
