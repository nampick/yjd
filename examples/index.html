yjd — framework integration dev tests

Local dev pages that mount the editor (from /dist) inside each framework. No build step — the framework loads from a CDN; the wrappers mirror the README and integration guides.

React →
Hook wrapper, controlled value ⇄ onChange, StrictMode-safe. Vue 3 →
Composition-API component with v-model two-way binding. Vue 2 →
Options-API component, v-model via value + input. AngularJS 1.x →
A directive with ng-model, manual bootstrap.

Modern Angular needs a build step, so there's no single-file demo — the standalone component is on the Angular guide.
