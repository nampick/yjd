# Rich Editor Icons

## Danh sách các file icon đã tạo:

### Text Formatting Icons

- `icon-bold.svg` - Icon cho chức năng in đậm (đã có sẵn)
- `icon-italic.svg` - Icon cho chức năng in nghiêng
- `icon-underline.svg` - Icon cho chức năng gạch chân
- `icon-strike.svg` - Icon cho chức năng gạch ngang

### Script Icons

- `icon-subscript.svg` - Icon cho chức năng chỉ số dưới (X₂)
- `icon-superscript.svg` - Icon cho chức năng chỉ số trên (X²)

### Color Icons

- `icon-color.svg` - Icon cho chức năng màu chữ
- `icon-background.svg` - Icon cho chức năng màu nền

### Action Icons

- `icon-undo.svg` - Icon cho chức năng hoàn tác
- `icon-redo.svg` - Icon cho chức năng làm lại
- `icon-more.svg` - Icon cho chức năng tùy chọn thêm

## Hướng dẫn sử dụng:

1. Thêm code SVG vào từng file tương ứng
2. Đảm bảo các SVG có kích thước phù hợp (thường 16x16 hoặc 20x20px)
3. Sử dụng màu sắc nhất quán với theme của editor
4. Test các icon trong toolbar để đảm bảo hiển thị đúng

## Ghi chú:

- Tất cả các file đã được tạo sẵn với comment placeholder
- User cần thêm code SVG thực tế vào từng file
- Các icon sẽ được sử dụng trong `toolbar.js` module
