  1. Font Size Button: Look for the "Size" button in the toolbar (should show "16px" by default)
  2. Select Text: Highlight some text in the editor below
  3. Choose Size: Click the font size dropdown and select a size (10px - 36px)
  4. Visual Preview: Each dropdown item shows the actual font size
  5. Custom Size: Try "Custom Size..." for manual input (e.g., 40px, 1.5em, 120%)
  6. Button Updates: The button text should update to show current font size
  7. Multiple Sizes: Apply different sizes to different text selections
